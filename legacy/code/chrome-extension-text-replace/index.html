A Chrome extension that replaces instances of the word 'females' with 'women' on twitter. Why? I personally don't like it for multiple reasons. Some of which are outlined here.

This was used to teach a 'making a chrome extension workshop' at Afrotectopia. You can use this as a template to make text replacer extensions.

If you're looking for inspiration, here's is a curated list of "activist" themed chrome extensions.
